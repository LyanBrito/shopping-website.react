
# 👗 Fashion - LandingPgage

Este é um projeto de landing page desenvolvido utilizando **React**, **Styled Components** e **Biome**, com foco em performance, modularidade e manutenibilidade.

## 🎨 Design no Figma

O design desta landing page foi planejado e prototipado utilizando o [**Figma**](https://www.figma.com/design/TP0gj5mPhrkKZT0MDwEmTj/Shopping-Website-(Community)?m=auto&t=eYP24aV9WFGGRt6c-6)

## 📁 Estrutura do Projeto

A estrutura do projeto segue boas práticas de organização:

```sh
    # Estrutura do Projeto

src/
├── assets/ 
│   ├── imgs/                       # Imagens PNG e JPG
│   └── svg/                        # Imagens SVG
├── components/
│   ├── Component/
│   │   ├── index.jsx               # Componente
│   │   └── styles.js               # Estilo do Componente
│   └── App.jsx                     # Aplicação

```

## 🛠️ Ferramentas Utilizadas

- **React:** Biblioteca JavaScript para criar interfaces de usuário.
- **Styled Components:** Estilização de componentes de forma declarativa.
- **Biome:** Ferramenta para formatação e linting de código.

## 🏗️ Instalação

1. Clone o repositório:
   ```bash
   git clone https://github.com/LyanBrito/shopping-website.react
   ```
2. Instale as dependências com o gerenciador de pacotes de sua preferência:
   ```bash
   cd shopping-website
   pnpm install
   # ou
   npm install
   # ou
   yarn install
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   pnpm start
   # ou
   npm start
   # ou
   yarn start
   ```

## ⚙️ Comandos

| Comando                   | Ação                                      |
| :------------------------ | :---------------------------------------- |
| `pnpm install`            | Instala as dependências                   |
| `pnpm start`              | Roda o projeto localmente                 |
| `pnpm build`              | Constrói a versão de produção             |
| `pnpm run format`         | Formata o código.                         |
| `pnpm run lint`           | Indentifica problemas de linting e estilo |
| `pnpm run check`          | Analisa o código sem alterações           |
